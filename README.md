## ✨ Features

| Feature              | Description                               |
| -------------------- | ----------------------------------------- |
| ⚡ Loading Screen    | Terminal boot animation on first visit    |
| 🖱️ Custom Cursor     | Yellow dot + ring that follows your mouse |
| 📊 Progress Bar      | Scroll progress indicator at top          |
| 🔢 Animated Counters | Stats count up when scrolled into view    |
| 🎞️ Scroll Animations | Sections fade in as you scroll            |
| 💡 Skill Tag Glow    | Yellow glow effect on hover               |
| 📧 Contact Form      | EmailJS powered — messages go to Gmail    |

## 🚀 Getting Started

```bash
# Clone the repo
git clone https://github.com/AnuragYadav12/personal-spotlight.git

# Navigate into it
cd personal-spotlight

# Install dependencies
npm install

# Run locally
npm start
```

Opens at `http://localhost:3000`

---

## ✏️ Customize

All content lives in one file:

```bash
src/data/portfolio.js      # Edit your name, bio, projects, links
src/components/About.jsx   # Swap in your photo
public/photo.jpg           # Your profile photo
public/Anurag_Yadav_CV.pdf # Your resume
```

---

## 📦 Deploy to Vercel

```bash
# Push to GitHub
git add .
git commit -m "my portfolio"
git push

# Then go to vercel.com → Import repo → Deploy
# Live in 60 seconds ✓
```
